9.Margin

Margins collapes with each other. So if the bottom and top margins are same then no difference will be noticed. But if they are different then the heighest value of the margin will be applied.
Block level elements are suitable for every types of margin but in the case of inline elements the top and bottom margins are not applicable. Only left and right margins are applibale. In this case the inline element should be changed in block level element using display: inline-block;.
